About this site

A few different technologies made this site possible.

Sass

Sass is a CSS extension language which introduces features such as variables, functions, and more advanced nesting into CSS.

Normalize.css

Normalize.css automatically applies certain preset stylings to account for browser stylesheets so websites look the same across browsers.

Modernizr

Modernizr is a JavaScript library which detects browser features and applies classes to the HTML element based on whether or not specified features are availible. This allows for CSS and JavaScript to be targeted based on feature support.

WebP

WebP is an image format developed by Google which uses significantly less data than JPEG and PNG formats while preserving quality. Modernizr was used to ensure that WebP images are only served to the browsers that support it, Chrome and Opera.

Adobe TypeKit

Adobe TypeKit is a web font service which provides the typefaces used on this site.

Git / GitHub

Git is a version control system which ensures that old code is never lost. Old versions can be rolled back to with a simple command, and code can be branched off for especially large changes to keep the codebase clean. GitHub allows for Git projects to be backed up to the web and worked on by multiple people seamlessly.

Google Web Starter Kit

Google's Web Starter Kit is a build system which includes the Sass compiler, an HTTP server, and a file optimizer. The Sass compiler converts Sass to valid CSS, the HTTP server which reloads automatically when files are saved and syncs scrolls and clicks across devices, and the file optimizer minifies HTML, CSS, and JavaScript, strips image metadata, and automatically prefixes the CSS files.

Other Software

Additional software used in making this site.

Adobe Photoshop

Photoshop was used to optimize the file size and crop certain assets.

Adobe Illustrator

Illustrator was used to design the logo.

Sublime Text 3

Sublime Text was the text editor used to develop the HTML and CSS.

Google PageSpeed Insights

Google PageSpeed Insights were used to check for issues resulting in poor performance or user experience.
